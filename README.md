# Merge-sort-youtube-tutorial
My version of code that I learnt from Youtube tutorials

## Опис
Цей код  також реалізує алгоритм Merge Sort — це метод сортування, який ділить масив на дві частини, рекурсивно сортує кожну з них, а потім об’єднує дві вже відсортовані половини в один масив. Функція merge(left, right) бере два відсортовані масиви й по черзі додає в результат менші елементи з кожного масиву. Функція mergeSort(array) спочатку ділить масив приблизно навпіл, викликає саму себе для кожної половини, а потім з’єднує їх за допомогою merge. Так масив поступово розбивається на дуже маленькі частини, а потім збирається назад — уже в правильному порядку. У кінці програма виводить відсортований масив. Проте я вивчила з Ютуб відеороликів. Код трішки відрізняється від того, що ми вчили на уроці та змінені назви об'єктів. Особисто для мене він зрозуміліший і я вирішила добавити його сюди <3
